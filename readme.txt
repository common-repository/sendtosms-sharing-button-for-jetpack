Add a SendToSMS button to Jetpack Sharing

== Description ==

### Notice

Don't forget to install Jetpack!

[Jetpack](http://jetpack.me) is awesome! It's used globally. This plugin adds an option to the 
Jetpack sharing bar that lets users share your posts via SMS text messages, for free!

= Want to find out more? =

Visit us at [sendtosms.com](http://www.sendtosms.com).

== Installation ==

1. Upload the `sendtosms-jetpack-button` folder to the `/wp-content/plugins/` directory or install from the WordPress Plugins Directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the settings for your Jetpack plugin
4. Click 'Configure' under the 'Sharing' module 
5. The SendToSMS button should appear in the list of available sharing services.
6. Drag the SendToSMS button down to the list of enabled services.
7. Click 'Save Changes' at the bottom of the page.

== Frequently Asked Questions ==

= Will this cost me, or the SMS recipient, any money? =
There is no cost for sending the SMS message.  However, if the recipient's mobile plan charges
for SMS messages received, they may be charged in accordance with that mobile plan.

= Will I start getting spam text messages? =
No.  We comply with the CAN-SPAM act, and will not send any promotional messages that you don't 
specifically ask for.  We will also never sell your information, including any mobile numbers you
provide, to third parties.
